üîÅ 4. File Methods Summary
Method	Description
read()	Reads entire file
read(n)	Reads n characters
readline()	Reads one line
readlines()	Returns list of lines
write(str)	Writes string
writelines()	Writes a list of strings
seek(n)	Moves cursor to byte n
tell()	Returns current cursor position
close()	Closes the file

python
Copy
Edit

    data = img_file.read()

with open("copy.png", "wb") as copy_file:
    copy_file.write(data)
c. Using seek() and tell()
python
Copy
Edit
with open("example.txt", "r") as file:
    print(file.read(5))
    print(file.tell())   # Where we are in file
    file.seek(0)         # Move to start
    print(file.read(5))

Copy

def read_large_file(filename):
    with open(filename, 'r') as f:
        while chunk := f.read(1024):  # 1KB at a time
            print(chunk)

read_large_file("largefile.txt")
